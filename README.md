## 项目结构

```
AlgoTypst/
├── main.typ          # 主入口文件
├── lib.typ           # 共享样式与组件库（封面、边注、代码块等）
├── chaptersTypst/    # 各章节 Typst 源文件（chapter1.typ ~ chapter11.typ）
├── assets/           # 插图与图片资源
├── slides/           # 课程讲义 PDF
└── main.pdf          # 编译输出
```

## 构建

安装 [Typst](https://github.com/typst/typst) 后运行：

```bash
typst compile main.typ
```
